🔄 完全重启开发服务器步骤
================================

1. 停止当前服务器
   - 在运行 npm run dev 的终端窗口
   - 按 Ctrl + C
   - 确保完全停止（看到命令提示符）

2. 清除 Vite 缓存（重要！）
   - 删除 node_modules/.vite 目录
   
   PowerShell 命令：
   Remove-Item -Recurse -Force node_modules\.vite -ErrorAction SilentlyContinue

3. 重新启动开发服务器
   npm run dev

4. 完全刷新浏览器
   - 按 Ctrl + Shift + R（Windows）
   - 或 Cmd + Shift + R（Mac）
   - 或打开开发者工具 -> 右键刷新按钮 -> 清空缓存并硬性重新加载

5. 验证配置
   - 打开浏览器控制台（F12）
   - 查看输出应该是：
     [Config] 🔧 开发环境配置: {
       baseURL: 'https://tbnx.plus7.plus/',  // ✅ 没有 = 号
       model: 'deepseek-reasoner',
       hasKey: true,
       keyPreview: 'sk-wOAmGmU...'
     }
