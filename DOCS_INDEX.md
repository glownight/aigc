## 💡 使用技巧

### 搜索文档

```bash
# 全文搜索
grep -r "关键词" *.md

# 搜索特定文档
grep "关键词" ARCHITECTURE.md
```

### 生成目录

大部分 Markdown 编辑器都支持自动生成目录：
- VS Code: Markdown All in One 插件
- GitHub: 自动渲染目录
- Typora: 自动目录功能

### 导出 PDF

使用工具导出 PDF：
```bash
# 使用 pandoc
pandoc README.zh-CN.md -o README.pdf

# 或使用 Markdown 编辑器的导出功能
```
